Hej jag heter Patrik Youssef och är en programerare med en hög grad av kompetens inom JavaScript, CSS3, Jquery, HTML samt grundläggande förståelse av ramverks som react. Nedan kan du se ett fåtal av mina tidigare slutförda projekt.

Bilder på tidigare projekt

beskrvining av tidigare projekt

  • Sten Sax Påse: en responsive webbapplikation som tillåter användaren spela sten sax påse med datorn som motståndare.
  • Memory: en responsiv webbapplikation som använder javascript för att återskapa det traditionella brädspelet memory.
  • Spel lista: en responsive webbapplikation som erbjuder en hög grad av interatkivitet och interaktion med DOM. Användare kan lägga till, ta bort samt filtera bort spel från listan.
  • Bilaffär: en responsive hemsida för en fiktionell bilaffär som är anpassad för att kunna vissas på mobiler, surfplattor samt desktops. Sidan använder javascript och jquery för att bland annat skapa en rotation av bilder i ett galleri, öka interaktivitet genom att bilder regerar på mouseover event samt att navbar kan gömas eller vissas om viewport har dimensionerna av en mobil enhet.
  • Väder : En responsive webbaplikation som använder Asynchronous JavaScript And XML för att hämta data i JSON format från openweathermaps api.
  • Klocka: En simpel och elegant responsive klocka som skapats genom html, css och javascript.
  • Konverterare: En webbapplikation som konverterar km/h till miles per hour.
  • Interaktiv sida: en responsive webbapplikation som använder ramverket react. I appen kan användare lägga till egna färger och ändra färgen på header, ta bort samtliga färger samt flyta på en cirkel från ena änden av skärmen till den andra.
